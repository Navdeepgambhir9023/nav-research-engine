## System Boundaries

### Inside the Harness

The harness owns and maintains:

1. **Knowledge Base**: All structured knowledge, entity schemas, and the knowledge graph
2. **Adaptive Discovery**: LLM-driven clarification of research queries
3. **Execution Layer**: Subagent/workflow orchestration
4. **Output Contract**: Standardized artifact formats
5. **Domain Config**: Domain-specific schemas and taxonomies

### Outside the Harness

The harness consumes but does not own:

1. **Domain Data Sources**: User-provided project files, external data, research
2. **LLM Providers**: Claude, Gemini, or any AI reasoning provider
3. **User Context**: Project files, existing research, domain knowledge
4. **Human Reviewers**: Subject matter experts who provide judgment
