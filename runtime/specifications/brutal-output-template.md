## 5. Source Citation Format

**Inline citations:**
```
According to [SRC-001], the market shows [X].
Multiple sources [SRC-001], [SRC-002] confirm [Y].
[SRC-003] contradicts [SRC-004] on [Z].
```

**Source ID format:**
- `SRC-XXX` where XXX is a sequential number
- `SRC-001`, `SRC-002`, etc.

**Source metadata:**
```
| ID | Title | URL | Type | Credibility | Last Verified |
|----|-------|-----|------|-------------|---------------|
```

**Source types:**
- `web` - Public web source
- `primary` - Primary research/interview
- `confidential` - Confidential source (anonymous)
- `inferred` - Inferred from indirect evidence

## 6. Completeness Checklist

Before publishing any output, verify:

- [ ] Executive Summary with confidence badge
- [ ] At least 3 Key Findings with sources
- [ ] Competitive Landscape with verification status
- [ ] Market Analysis with known limitations
- [ ] HONEST ASSESSMENT section (never skip)
- [ ] WHAT THIS CANNOT DETERMINE section (never skip)
- [ ] All inline citations have corresponding source entries
- [ ] Brutality Index rating included
- [ ] No unsupported claims (all statements cited or marked as inference)
